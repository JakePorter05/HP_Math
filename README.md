# 🧙‍♂️ HP Math  
*A Wizard‑Themed Console Math Revision Game*

## ✨ Overview
HP Math is a simple console‑based math revision game created as part of the **C# Academy** learning path. It provides a light wizard‑school theme to make practicing arithmetic more enjoyable, without involving combat or spell duels. Players choose a math category, answer randomly generated questions, and receive immediate feedback.

This project reinforces core C# fundamentals such as loops, methods, user input handling, and random number generation.

---

## 🧮 How the Game Works
- You are greeted with a wizard‑themed welcome message.  
- Choose the type of math you want to practice:  
  - Addition  
  - Subtraction  
  - Multiplication  
  - Division  
- The game generates random problems based on your selection.  
- Enter your answer directly into the console.  
- The game tells you whether you were correct and tracks your score.  
- Continue practicing until you choose to exit.

---

## 🎯 Learning Goals
This project was built to strengthen:
- Basic arithmetic practice  
- Working with variables and data types  
- Using loops for repeated tasks  
- Structuring logic with conditionals  
- Creating reusable methods  
- Handling user input safely  
- Generating random numbers
